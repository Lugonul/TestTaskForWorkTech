# Library Management System

Веб-приложение для управления библиотекой с использованием современного стека технологий Java.

# О проекте

Library Management System - это REST веб-приложение для управления книгами, авторами и жанрами в библиотеке. Приложение предоставляет полный CRUD функционал, поиск, авторизацию пользователей и документирование API.

# Технологии

Spring Boot для создания REST API.

Hibernate для работы с базой данных PostgreSQL.

JUnit и Mockito для написания тестов.

Git для контроля версий.

Maven/Gradle для сборки проекта.

Swagger для документирования API.


# Функционал

Добавление, удаление и редактирование книг.

Поиск книг по названию, автору или жанру.

Регистрация и авторизация пользователей.

Логирование действий пользователей.

Документирование API с использованием Swagger.

Добавление, удаление и редактирование авторов и жанров только для роли admin
